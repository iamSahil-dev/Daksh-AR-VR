# EduARtisan - DAKSH Horizon Skills

An AI + AR-powered, offline-first skill development ecosystem that transforms any smartphone into a virtual skill laboratory for vocational education in India.

## üéØ Problem Statement

India's vocational education system faces a deep-rooted accessibility crisis. Millions of students in rural and semi-urban regions lack exposure to practical, hands-on training due to:
- Absence of physical labs
- Limited internet access
- Shortage of trained instructors
- English-centric e-learning platforms that fail to engage low-literacy learners

Existing e-learning solutions are heavily internet-dependent and fail to prepare youth for the evolving demands of Industry 4.0.

## üí° Solution: EduARtisan

An AI + AR-powered, offline-first skill development ecosystem that transforms any smartphone into a virtual skill laboratory. Learners can practice industrial operations such as motor assembly, solar panel wiring, or machine maintenance through 3D AR modules, guided by a voice-enabled AI tutor in regional languages.

## ‚ú® Key Features

- **Adaptive AR Learning** - Real-time AI feedback personalizes training difficulty
- **Multilingual Voice Guidance** - Teaches through regional language voice commands for inclusivity
- **Offline Edge AI** - All modules function without internet; data syncs automatically when online
- **Reality Sync Mode** - AR overlays guide users on real machinery, bridging virtual and real learning
- **Community Skill Network** - Gamified certification system that motivates and connects learners

## üöÄ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/iamSahil-dev/Daksh-AR-VR.git

# Navigate to project directory
cd daksh-horizon-skills-main

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at `http://localhost:8080`
